# ML Strategy Unit Tests

This directory contains unit tests for ML trading strategies.

## Test Files

- `test_base_strategy.py` - Tests for the base ML strategy functionality
- `test_signal_strategies.py` - Tests for strategies that consume ML signals

## What Belongs Here

Unit tests for:
- Strategy initialization and configuration
- Strategy lifecycle methods
- Order placement logic based on ML signals
- Risk management in ML strategies
- Position management
- Strategy state management
- Signal consumption and processing
