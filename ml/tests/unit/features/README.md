# ML Feature Engineering Unit Tests

This directory contains unit tests for ML feature engineering and validation.

## Test Files

- `test_feature_engineering.py` - Tests for feature computation and transformation
- `test_feature_validation.py` - Tests for feature validation and quality checks
- `feature_parity/` - Tests ensuring training/inference feature parity

## What Belongs Here

Unit tests for:
- Feature computation algorithms
- Feature transformation pipelines
- Feature validation rules and checks
- Feature caching and persistence
- Training/inference feature consistency
- Feature quality metrics
- Data preprocessing and normalization
