# ML Data Processing Unit Tests

This directory contains unit tests for ML data loading and processing.

## Test Files

- `test_data_loader.py` - Tests for data loading functionality
- `test_data_structure.py` - Tests for ML-specific data structures

## What Belongs Here

Unit tests for:
- Data loading from various sources
- Data format validation and parsing
- Data structure creation and manipulation
- Data pipeline components
- Data caching and storage
- Data transformation utilities
- Data quality checks and validation
