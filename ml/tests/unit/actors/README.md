# ML Actor Unit Tests

This directory contains unit tests for ML actors in the Nautilus ML system.

## Test Files

- `test_base_actor.py` - Tests for the base ML actor functionality
- `test_inference_actor.py` - Tests for actors that perform ML inference
- `test_signal_actor.py` - Tests for actors that generate trading signals from ML models

## What Belongs Here

Unit tests for:
- Actor initialization and configuration
- Actor lifecycle methods (on_start, on_stop, on_reset)
- Data processing within actors
- Signal generation and publishing
- Actor state management
- Error handling in actor operations
