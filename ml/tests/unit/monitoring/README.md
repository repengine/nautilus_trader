# ML Infrastructure Unit Tests

This directory contains unit tests for ML infrastructure components.

## Test Files

- `test_dashboard_factory.py` - Tests for monitoring dashboard creation
- `test_grafana_client.py` - Tests for Grafana integration
- `test_optuna_optimizer.py` - Tests for hyperparameter optimization

## What Belongs Here

Unit tests for:

- Monitoring and dashboard functionality
- External service integrations (Grafana, MLflow, etc.)
- Hyperparameter optimization
- Model deployment infrastructure
- Logging and metrics collection
- Service configuration and health checks
- Infrastructure utilities and helpers
