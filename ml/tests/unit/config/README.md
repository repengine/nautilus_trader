# ML Configuration Unit Tests

This directory contains unit tests for ML configuration classes and validation.

## Test Files

- `test_config_classes.py` - Tests for general ML configuration classes
- `test_ml_actor_config.py` - Tests for ML actor-specific configuration

## What Belongs Here

Unit tests for:
- Configuration class validation
- Configuration serialization/deserialization
- Configuration defaults and overrides
- Configuration schema validation
- Actor-specific configuration parameters
- Model configuration validation
- Training configuration parameters
